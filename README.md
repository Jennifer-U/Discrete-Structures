# Discrete Structures - Winter 2024

Course covering core topics in discrete mathematics and their computational applications.

## Course Overview
Exploration of algorithms, recursion, recurrences, asymptotics, relations, graphs, and trees with practical implementations.

## Topics Covered

**Foundations**
- Logic and Truth Tables
- Boolean Algebra
- Basic Mathematical Structures

**Combinatorics**
- Permutations and Combinations
- Evaluating and Ranking Functions

**Algorithms & Analysis**
- Array Search Algorithms (empirical performance analysis)
- Asymptotic Analysis
- Algorithm Complexity

**Recursion & Recurrences**
- Induction and Recursion
- Divide-and-Conquer Recurrences
- Linear Homogeneous Recurrences

**Probability**
- Discrete Probability Theory

**Graph Theory**
- Graphs and Graph Algorithms
- Trees and Tree Algorithms

## Implementation
All assignments completed in **Python** with:
- Algorithm implementations
- Performance benchmarking and empirical analysis
- Data visualization and plots
- Graphical representations of results
